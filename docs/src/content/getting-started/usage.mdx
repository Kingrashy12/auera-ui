<Details id="globals">
  <HeaderLabel id="globals">Globals</HeaderLabel>
  <p className="text">Since AueraUI is built to function globally, you can control the look and behavior of your app using the <CodeTag>Provider</CodeTag> and hooks.</p>
</Details>

<Details id='theme'>
 <HeaderLabel id="theme" as='h3'>Theme</HeaderLabel>
 <p className="text">AueraUI allows you to customize the look and feel of your application with its flexible theming system. 
 The <CodeTag>ThemeProvider</CodeTag> component enables global theme management, allowing you to apply consistent styling across your app.</p>
</Details>

<Details id="default-theme">
 <HeaderLabel id="default-theme" as='h3'>Default Theme</HeaderLabel>
<p className="text">By default, AueraUI detects the user's system theme and applies the appropriate theme. This provides a seamless experience for users,
 as the UI will automatically align with their system settings.</p>
</Details>

<Details id="toggle-theme">
 <HeaderLabel id="toggle-theme" as='h3'>ToggleTheme</HeaderLabel>
<p className="text">
AueraUI provides a toggle function that allows you to manually switch between the user's system theme and AueraUI's theme set.
 The toggle function has two main options:
 </p>

 <ul className="text font-inter list-disc pl-3 gap-4 flex flex-col">
    <li>
        <CodeTag>toggleTheme.system</CodeTag>: Switches to the user's system theme based on the user's operating system settings.
    </li>
    <li>
        <CodeTag>toggleTheme.main</CodeTag>: Switches to AueraUI's predefined theme set, 
    </li>
  </ul>
  <p className='text'>
  This toggleTheme functionality is useful when you want to give users the ability to manually switch between system preferences and the app's theme.
  </p>
  <h4 className="text-primary font-inter font-semibold">
  Example Usage:
  </h4>
  <CodeBlock fileName='Demo.tsx' lg='tsx' code={`import { useTheme, Button } from 'auera-ui';

export default function Demo() {
  const { toggleTheme } = useTheme();

  return (
    <div>
      <Button onClick={() => toggleTheme.system()}>Use System Theme</Button>
      <Button onClick={() => toggleTheme.main()}>Use AueraUI Theme</Button>
    </div>
  );
}`} />
</Details>
